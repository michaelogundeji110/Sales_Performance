# Sales Analysis with Excel and Power BI

## Project Objective
To analyze sales performance and identify trends, top-performing regions, and key revenue drivers.

## Tools Used
- Microsoft Excel (data cleaning, pivot tables)
- Power BI (data modeling, DAX, visualization)

## Dataset
- Source: Sample sales dataset
- Records: Monthly sales by region and product

## Key Tasks Performed
- Cleaned and structured raw data in Excel
- Created pivot tables and charts
- Built an interactive Power BI dashboard
- Analyzed regional and product performance

## Key Insights
- Top 3 regions contributed over 60% of total sales
- Q4 showed the highest sales growth
- A small number of products generated most revenue

## Files in this Repository
- `data/raw_sales_data.xlsx`
- `excel/sales_analysis.xlsx`
- `powerbi/sales_dashboard.pbix`
